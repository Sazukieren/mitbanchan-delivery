# 🍱 밑반찬 배달 앱

삼전복지관 밑반찬 배달 봉사 전용 앱입니다.

## 📱 앱 설치 방법

### iPhone (iOS)
1. Safari로 접속: https://Sazukieren.github.io/mitbanchan-delivery/
2. 하단 공유 버튼(□↑) 탭
3. "홈 화면에 추가" 탭
4. "추가" 탭 → 완료!

### Android
1. Chrome으로 접속: https://Sazukieren.github.io/mitbanchan-delivery/
2. 상단 설치 배너 "설치" 탭
3. 또는 메뉴(⋮) → "앱 설치"

## 💡 자주 묻는 질문
- **데이터가 사라졌어요** → 설정 → 데이터 내보내기로 백업하세요
- **지도가 안 떠요** → 인터넷 연결 확인 후 새로고침
- **오프라인에서도 돼요?** → 네, 한 번 열면 인터넷 없어도 작동합니다

## 🔧 기능
- 배달처 목록 관리 및 완료 체크
- 카카오맵으로 위치 확인
- 방문 사진 촬영 및 저장
- 배달 통계 및 기록
- 오프라인 작동 (PWA)
